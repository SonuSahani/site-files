Detailed Itinerary

Arrival → Hotel check-in → Howrah Bridge, Kumartuli, College Street, Sovabazar Rajbari, Prinsep Ghat, Rabindra Sadan, Nandan → Evening visit to Technicians Studio (live sets) → Overnight Kolkata.

Full morning at NT1 / Indrapuri Studio (guided tour + live shooting if allowed) → Victoria Memorial, Esplanade, Park Street, Metro Cinema → Drop to airport/station. Tour ends.
